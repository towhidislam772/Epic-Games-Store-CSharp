
# Project Title

🎮 Epic Games Store Clone (C#)
## Features


🧑‍💻 User Registration & Login

🛒 Game Browsing & Cart System

🔍 Search Functionality

📦 Game Details View

🎨 Modern UI with custom styling

💾 Local data storage using JSON or flat files
